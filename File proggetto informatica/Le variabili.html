LE VARIABILI

Una variabile in C è uno spazio di memoria che contiene un valore. Una variabile ha un nome che viene usato nella programmazione per riferirsi a un certo particolare spazio di memoria. Sono dei “contenitori” definiti da un tipo e da un nome. Vanno dichiarate all’inizio del programma, quindi deve essere detto al compilatore il tipo della variabile, ed infine va inizializzata, ovvero le deve essere assegnato un valore.

Tipi di variabili

Ad ogni variabile in C deve essere associato un tipo, quindi quando si dichiara una variabile bisogna specificare sempre di che tipo è. In C le variabili possono essere di 6 tipi base:
int (numero intero)
char (carattere)
float (numero reale)
double (numero reale lungo)
long (numero intero lungo)
short (numero intero corto)

Dichiarare una variabile

Per essere usata una variabile va prima dichiarata:
La dichiarazione di una variabile è del tipo: “tipo nomevariabile;” dove “nomevariabile” è il nome della variabile e “tipo” e il tipo della variabile.

int numero; /*per dichiarare un numero intero.*/:

Si possono anche dichiarare più variabili su una singola riga.

int a,b;

Le variabili si possono inizializzare anche durante la dichiarazione.

int a = 1;

Finché la variabile non è inizializzata il suo valore è indeterminato ed essendo che non tutti i compilatori agiscono nello stesso modo, è sempre meglio inizializzare le variabili prima dell’uso, in modo tale da rendere il programma compatibili con il maggior numero possibile di compilatori.

int a, b = 1; /*si creano due variabili, ma solo la seconda è inizializzata*/
int a,b,c = 1; /*tutte le variabili vengono inizializzate con lo stesso valore*/

Nomi delle variabili

Il nome di una variabile viene scelto dal programmatore, e tecnicamente si chiama "identificatore". La scelta di un particolare nome non ha influenza sull'esecuzione del programma, tuttavia, vanno rispettate delle regole:
-Due variabili non possono avere lo stesso nome
-Si possono usare sia le lettere che i numeri, ma il primo carattere deve essere una lettera
-Il C è case sensitive, quindi le maiuscole sono distinte dalle minuscole
-Le parole riservate come if, int, etc… non possono essere usate per i nomi delle variabili.

ESEMPIO COMPLETO

#include int main
{
int n = 45;

float N_decimale = 54.78;
double N_decimale_con_più_spazi_di_memoria = 34.567;
char carattere = 'a';

printf("La variabile intera è uguale a %d", n);

/*Si può cambiare il valore della variabile anche dopo averla dichiarata*/

n = 16;
N_decimale= 33.9;
N_decimale_con_più_spazi_di_memoria= 5679.468;
carattere = 'b';
}
